# Portfolio

## Usage

### Setup

```bash
git lone https://github.com/BlaxBerry/portfolio.git
yarn
yarn run develop
```

### git commit 

- feat: (feature)
- fix: (fix bugs)
- docs: (documents)
- style:
- pref:
- merge:
- revert:

### How toupdate

```bash
yarn run build
yarn run preview
git add .
git cimmit -m "message"
git push
```
